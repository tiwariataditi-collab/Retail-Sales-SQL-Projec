# Retail Sales Analysis SQL Project

## Project Overview
* **Project Title:** Retail Sales Analysis
* **Level:** Beginner
* **Database:** retail_project_db

This project is designed to demonstrate SQL skills and techniques typically used by data analysts to explore, clean, and analyze retail sales data. It involves setting up a retail sales database, performing basic data cleaning, and answering specific business questions through SQL queries.

---

## Objectives
1. **Database Setup:** Create and populate a retail sales database with the provided dataset.
2. **Data Cleaning:** Identify and remove any records with missing or null values, and standardize date fields.
3. **Business Analysis:** Use standard SQL queries to answer specific business questions and derive store insights.

---

## Business Questions to Solve

* **Q1:** What is the highest (maximum) and lowest (minimum) single sale amount in our store?
* **Q2:** How many total items (quantity) were sold in the 'Electronics' category?
* **Q3:** List the top 3 most expensive items based on their price per unit.
* **Q4:** What is the average dollar amount spent per transaction by 'Female' customers?
* **Q5:** Find all sales transactions where a customer bought exactly 4 units of any item.
* **Q6:** Which product categories have achieved a total revenue of more than $50,000?
* **Q7:** What is the total number of orders placed during the year 2023?
* **Q8:** Find the average profit made on a single transaction across the entire store.
* **Q9:** Group our transactions by age to see how many orders come from each individual age.
* **Q10:** What is the total revenue generated by 'Clothing' sales to 'Male' customers?

---

## How to Run This Project
1. Open **XAMPP Control Panel** and start Apache and MySQL.
2. Load the dataset into your MySQL server instance using your Python ETL script.
3. Open **VS Code**, connect your SQLTools extension, and run the queries individually to analyze the reports.
